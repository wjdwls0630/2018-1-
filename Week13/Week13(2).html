Variable 선언 시 주의점

  1. 변수명 첫 글자로는 $, _ , 영문자만 올 수 있다.
  2. 변수명 첫 글자 다음은 영문자, 숫자, $ , _ 만 올 수 있다.
  3. 변수명으로는 예약어(document,location,window 등) 을 사용할 수 없다. 예약어란 자바스크립트에서 사용 중인 단어들
  4. 변수명을 지을 때는 되도록 의미 부여!
  5. 변수명을 사용 할 때는 대,소문자 구분!

String Concatenation

Two of Data Type(String, Number) can concat or apply addition operator

String + Number can be possible
String + Number becomes String
Use +operator or concat method

Tip :

concat method 와 + 연산자의 결과물은 같지만 둘의 작동 방식은 다르다

concat method 는 a.concat(b)를 하면 결과물을 새로운 string으로 생성해 결과물 c 로 반환 한다.
따라서 계속 문자열을 추가할 때마다 변수가 가리키는 변수의 id값이 계속 변한다.

+는 StringBuilder 클래스로 새로운 인스턴스를 만들어 append로 문자열을 추가한 뒤 toString()으로 타입변환을 한 후 리턴한다
따라서 추가된 결과물이 가리키는 id값은 +연산자 앞에 있는 객체의 id값과 같다.
